I
Comparador de Preços — "O Google Flights do vinho"
Meses 1–2 · Custo estimado: R$ 0–2.000
▾

Um site simples onde o usuário pesquisa um vinho (por nome, uva ou região) e vê os preços em tempo real nas principais lojas brasileiras. Sem estoque, sem logística, sem risco legal.

Como funciona

Scraping de Evino, Wine, Vivino, Divvino
↓
Base de dados normalizada de vinhos
↓
Usuário pesquisa → vê ranking de preços
↓
Botão "me avise se baixar" → captura email

O que você ganha

Base de usuários, dados de demanda ("qual vinho todo mundo quer?"), prova de conceito do produto sem investimento.

Métrica de sucesso

500 usuários cadastrando alerta de preço em 60 dias → demanda real comprovada.

Checklist de lançamento

  • Scraper das 5 maiores lojas online de vinho do Brasil
  • Banco de dados de vinhos com nome normalizado (evitar "Catena Zapata" vs "DV Catena")
  • UI simples de busca + comparativo de preços
  • Alerta de preço por email (SendGrid grátis até 100/dia)
  • Pixel do Meta + GA4 para rastrear comportamento
  • Landing page com "lista de espera" para o clube
II
Clube de Assinatura — "A curadoria que você não tem tempo de fazer"
Meses 3–6 · Receita recorrente começa aqui
▾

Com a base de usuários do comparador, você lança o clube. O parceiro sommelier seleciona 2–3 vinhos por mês, você cuida da logística com um operador 3PL (terceirizado) e cobra uma assinatura mensal.

Modelos de assinatura

PlanoConteúdoPreço/mêsMargem estimada
Explorador2 garrafas curadasR$ 149~35%
Sommelier3 garrafas + notas de degustaçãoR$ 229~40%
Colecionador6 garrafas + vinhos raros + frete grátisR$ 429~45%

Vantagem competitiva

Você sabe exatamente quais vinhos os usuários procuraram no comparador. A curadoria do clube não é um chute — é baseada em demanda real da sua própria base.

Logística sem dor de cabeça

Use operadores 3PL especializados em bebidas (como Patrus ou Jadlog Shops). Você não toca no estoque — só gerencia os pedidos pela plataforma.

III
Compra Coletiva — "O diferencial que ninguém tem"
Meses 6–12 · O coração do produto
▾

Aqui está a sua ideia original, adaptada legalmente. Um importador habilitado parceiro fecha lotes quando a demanda mínima é atingida. Você é a plataforma que agrega essa demanda.

O fluxo completo da compra coletiva

Etapa 1
Curadoria — Sommelier identifica um vinho argentino de alta qualidade com grande diferença de preço BR/AR
Etapa 2
Oferta — App abre uma "rodada" com preço alvo, quantidade mínima e prazo (ex: 30 dias para 200 garrafas)
Etapa 3
Intenção de compra — Usuários confirmam interesse e pré-autorizam o pagamento (não cobrado ainda)
Etapa 4
Ativação — Meta atingida → importador fecha o pedido, pagamentos são capturados
Etapa 5
Importação legal — Importador habilitado traz o lote com todos os impostos pagos, notas fiscais emitidas
Resultado
Entrega — Vinho chega com rastreamento. Preço final: ~40–60% menor que no varejo nacional

Exemplo real: DV Catena Adrianna Vineyard

Preço na Argentina (Mendoza)~R$ 80
Impostos de importação (~180%)~R$ 144
Frete + seguro por garrafa (lote 500un)~R$ 18
Margem da plataforma (12%)~R$ 29
Preço final ao usuário~R$ 271
Preço no varejo brasileiroR$ 420–480

→ Economia real de ~40% mesmo com todos os impostos pagos.

IV
Importação Própria — "Capturar a margem total"
Ano 2+ · Requer volume e capital de giro
▾

Com R$500k+ em GMV mensal, faz sentido habilitar o próprio CNPJ importador. Você deixa de pagar a margem do parceiro importador e captura mais ~15–20% por garrafa.

Habilitação de importador — o que é necessário

  • ✓
    CNPJ ativo com CNAE de importação de bebidas alcoólicas
  • ✓
    Habilitação no Siscomex (Receita Federal) — processo burocrático mas viável
  • ✓
    Registro no MAPA (Ministério da Agricultura) para bebidas alcoólicas
  • ✓
    Capital de giro para financiar o estoque em trânsito (30–90 dias)
  • ✓
    Despachante aduaneiro especializado em bebidas

Dica: Contrate um advogado especializado em comércio exterior desde a Fase 3. O custo (~R$3–8k) é irrisório comparado ao risco de errar no processo.

Arquitetura Técnica

Stack pensado para
crescer sem reescrever

Como você vai desenvolver solo inicialmente, a prioridade é velocidade de entrega e facilidade de manutenção — não a stack mais sofisticada.

Stack recomendada

Frontend
Next.js 14 (App Router) + Tailwind
SSR nativo para SEO (vinhos indexam bem no Google), deploy trivial na Vercel, um código para web e mobile via PWA
Backend
Next.js API Routes + tRPC
Evita manter um backend separado no MVP. Migra para microserviços depois se precisar.
Banco
PostgreSQL (Supabase) + Prisma ORM
Supabase dá auth, storage e realtime grátis no tier inicial. Prisma garante type-safety no acesso a dados.
Scraping
Python + Playwright + Redis Queue
Playwright lida com sites com JS. Redis gerencia a fila de scraping para não sobrecarregar os sites alvo.
Pagamentos
Stripe (global) + Pagar.me (BR)
Stripe para assinaturas (melhor UX), Pagar.me para PIX e boleto — essenciais no Brasil.
Email
Resend + React Email
Emails transacionais e de marketing. React Email permite criar templates em JSX — mesma linguagem do frontend.
Infra
Vercel (frontend) + Railway (scraper Python)
Vercel é grátis para projetos Next.js. Railway é simples para subir serviços Python com Redis.

Para o Pitch com Investidores

"O mercado brasileiro de vinhos cresce 18% ao ano. Ninguém ainda resolveu o problema de preço de forma coletiva e legal. Nós temos a tecnologia, o sommelier e o modelo."

Por que agora?

Pandemia criou uma nova geração de consumidores de vinho em casa. Classe média premium busca qualidade sem pagar preço de importadora tradicional.

O moat

Dados de demanda + relacionamento com importadores parceiros + reputação do sommelier. Cada rodada bem-sucedida fortalece os três.

A saída

Aquisição por Wine.com.br, Vivino ou grupo de bebidas (Ambev, por exemplo). Ou IPO se a escala justificar.

Parceiros estratégicos para buscar agora

  • Importador habilitado — procure na ABIA ou em grupos de importadores de SC/RS. Modelo de revenue share (sem exclusividade)
  • Vivino — têm programa de parceiros para apps de comparação. Acesso a base de dados de vinhos já normalizada
  • Sommelier certificado — seu parceiro. Credencial WSET mínimo nível 2 para dar credibilidade
  • Operador logístico 3PL de bebidas — essencial para clube de assinatura sem tocar em estoque
  • Advogado de comércio exterior — não pule essa etapa. R$5k preventivos valem mais que R$50k reativos
Próximos Passos

O que fazer
esta semana

Tudo começa sem investimento. A única coisa que você precisa agora é de tempo e disciplina para validar antes de construir.

⁂

Semana 1 — Validação da demanda (custo: R$ 0)

  • Criar uma landing page simples (Carrd.co ou Framer — grátis) com proposta de valor e formulário de interesse
  • Postar em 3 grupos de vinho no Facebook/WhatsApp perguntando: "você pagaria X por Y se chegasse R$80 mais barato?"
  • Entrevistar 10 consumidores de vinho sobre hábitos de compra e frustração com preços
  • Identificar 2–3 importadores de vinho em SC/RS e mandar mensagem de apresentação

Semana 2–3 — Primeiro protótipo técnico

  • Scraper básico da Evino e Wine.com.br com Playwright (Python)
  • Banco de dados no Supabase — tabela wines + price_snapshots
  • Página de busca Next.js simples — pesquisa por nome, retorna preços
  • Deploy na Vercel — URL pública para mostrar para possíveis parceiros

Semana 4 — Parcerias e legalidade

  • Abrir MEI ou LTDA (MEI não serve para importação — verifique com contador)
  • Consulta inicial com advogado de comércio exterior (1h = ~R$400)
  • Fechar parceria informal com importador para a fase de compra coletiva
  • Definir com o parceiro sommelier os 3 primeiros vinhos da curadoria
  • Aplicar ao Startup SC ou ACE Startups para mentoria gratuita
